# Persistent Storage 

## **User Story**
As a user,
I want my liked/saved recipes and folders to be stored in a database so that my data persists between sessions and I can access it anytime. 

## **Acceptance Criteria**
- Users can see their saved recipes after logging out and logging back into their account.
- Users can see their created folders after logging out and logging back into their account.
- Users can see their saved recipes after refreshing the website. 
- Users can see their created folders after refreshing the website.
